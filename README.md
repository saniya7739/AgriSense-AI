# AgriSense AI

## Description

AgriSense AI is an AI-powered farming assistant designed to empower farmers with intelligent tools for modern agriculture. The platform provides comprehensive support for crop management, disease detection, weather forecasting, market insights, government scheme access, and data policy compliance, all in multiple Indian languages.

## Features

### Core Modules
- **Dashboard**: Central hub for accessing all features and viewing key metrics
- **AI Kisan Mitra**: AI-powered assistant for personalized farming advice
- **Crop Management**: Library and tools for crop selection, planning, and monitoring
- **Disease Detection**: AI-based disease identification and treatment recommendations
- **Weather Advice**: Real-time weather forecasts and farming recommendations
- **Market Prices**: Current and historical market price information
- **Government Schemes**: Information and application assistance for govt programs
- **Farmer Community**: Platform for farmer-to-farmer knowledge sharing
- **Emergency Alerts**: Critical alerts for weather events, pest outbreaks, etc.
- **Request Pickup**: Logistics coordination for produce collection
- **Soil & Fertilizer**: Soil analysis and fertilizer recommendations

### Compliance Module
- **Policy Management**: Upload and manage PDF policy documents
- **Violation Detection**: Automated detection of policy violations
- **Human Review Workflow**: Interactive review system for violations
- **Audit Reports**: Professional compliance reports with export capabilities
- **Remediation Panel**: Actionable suggestions for resolving violations

### Additional Features
- **Multilingual Support**: Hindi, English, Bhojpuri, Marathi, Punjabi
- **Mobile-Responsive**: Optimized for mobile devices
- **Offline Capabilities**: Basic functionality without internet
- **Data Security**: Compliant with agricultural data policies

## Installation & Setup

No installation required. This is a client-side web application.

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for real-time features

### Running the Application
1. Clone or download the repository
2. Open `agrisense/index.html` in your web browser
3. Select your preferred language
4. Enter mobile number or continue as guest
5. Access the dashboard and explore features

## Usage Guide

### For Farmers
1. **Registration**: Enter mobile number for personalized experience
2. **Dashboard Navigation**: Access all features from the main dashboard
3. **Module Usage**: Each module provides specific agricultural assistance
4. **Language Selection**: Choose from 5 Indian languages
5. **Offline Mode**: Basic features work without internet

### For Administrators
1. **Compliance Monitoring**: Use the compliance dashboard
2. **Policy Management**: Upload and manage policy documents
3. **Report Generation**: Create audit reports and export data
4. **Violation Review**: Human oversight of automated detections
